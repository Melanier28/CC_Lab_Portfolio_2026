Project 6 Title

p5.js Sketch

Description

For my data portrait, I visualized how long I use my most frequently opened apps. Instead of creating a static chart, I decided to build an interactive phone interface in code. The phone screen displays my apps, and when you click on each one, it opens to reveal the amount of time I spend using it. This approach allowed me to turn personal data into something playful, dynamic, and relatable, while also practicing UI design and interactive coding.
